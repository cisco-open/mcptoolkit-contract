# Support

Thanks for using **mcpcontract**!

## Where to ask for help

- **Bug reports & feature requests**: open an issue using the
  [GitHub issue tracker](https://github.com/cisco-open/mcptoolkit-contract/issues).
- **Usage questions & discussion**: use
  [GitHub Discussions](https://github.com/cisco-open/mcptoolkit-contract/discussions).
- **Security issues**: please follow the disclosure process described in
  [SECURITY.md](SECURITY.md). Do **not** open a public issue.

## Documentation

- [README](README.md) — project overview and quick start
- [101 Tutorial](docs/101-tutorial.md) — short walkthrough
- [docs/users/](docs/users/) — user-facing guides, tutorials, and examples
- [docs/maintainers/](docs/maintainers/) — design notes and implementation history
- [CHANGELOG](CHANGELOG.md) — version history

## Response expectations

This project is maintained on a best-effort basis by the maintainers listed in
[MAINTAINERS.md](MAINTAINERS.md). We aim to acknowledge issues and pull requests
within 10 days, but there are no service-level commitments.
